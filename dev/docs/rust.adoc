== Build and Run Examples

[source, shell]
----
$ cargo run --example move_sprite > dev/logs/examples/move_sprite.log 2>&1
$ sudo apt install librust-wayland-sys-dev
$ sudo apt install librust-libudev-sys-dev
$ sudo apt install libxkbcommon-x11-dev
$ sudo apt install mesa-vulkan-drivers
----

`.cargo/config.toml`::
----
[target.'cfg(target_os = "linux")']
rustflags = ["-C", "link-arg=-fuse-ld=mold"]
----
